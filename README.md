Modular Open Suite of Standards (CMOSS). Universal Command and Control (C2) Interface (UCI) Standard. ARL Battlespace is a platform for developing novel AI for complex adversarial reasoning in MDO. Structured Threat Intelligence Graph (STIG). Real-Time Battle Field Management System. Extensible Battle Management Language. The Advanced Framework for Simulation, Integration, and Modeling (AFSIM). The CLARITY project consists in developing and structuring an ecosystem of major actors centered on the open-source MBSE solution. United Architecture Framework (UAF). Command and Control Simulation Interoperation (C2SIM). PlanMilT is a web application designed to assist in the planning of military operations at the tactical level. ODIN aims to become an alternative to industry-led commercial C2IS. C Model Developer CMD is an open-source C source code based environment for building simulations of systems described by time-based ODEs. Boomslang is a C2 simulator that provides an open-source tool to simulate new C2 doctrines and exercises. PythonPDEVS is a Python implementation of the Parallel Discrete Event System Specification (PDEVS) formalism. ModelicaDEVS is a free library for discrete-event modeling. Protocol for composing Distributed DEVS Models Across different Simulation Frameworks. The ARL Hierarchical MultiScale Framework (ARL-HMS) is a software library for development of multiscale models on heterogeneous high-performance computing systems. OpenMAS streamlines asynchronous Multi-Agent System

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
